- Discussions: [Reddit](https://www.reddit.com/r/adventofcode/)
- Language: [Rust](https://www.rust-lang.org/)

## How to Run
Run
```bash
cargo run --release --bin <xx>
```
and replace `<xx>` by the day number, e.g. `05`, or `17`.
